Online Food Ordering System which has following features
- Admin   
    - Add/Remove Restaurant
    - Add/Remove User
 
- Restaurant
    - See its order
    - Change Order Status to PREPARING, COMPLETED, DELIEVERED
    - Update Menu
    - Update Timing
- User
    - Order Food
    - See Details of All restaurant
